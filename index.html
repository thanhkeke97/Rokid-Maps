How It Works

Bluetooth-Powered Architecture

Two apps that talk to each other over Bluetooth SPP (Serial Port Profile)

Phone App

  • GPS tracking @ 1Hz
  • OSRM routing
  • Nominatim search
  • BT server
  • A2DP audio

Glasses App

  • Live rotating map
  • Turn-by-turn HUD
  • Route line overlay
  • Phone notifications
  • 3 layout modes
1

Search

Type an address or place name and pick from the results using OpenStreetMap data

2

Navigate

Routes are calculated with OSRM. If you go off-route, it automatically recalculates

3

Follow

Your glasses show a live rotating map with your route, step-by-step directions, and more

Powerful Features

Live Map Display

Rotates with your heading, renders dark-themed tiles with a green HUD overlay

Turn-by-Turn Directions

Shows current instruction, maneuver arrow, and distance to next turn

Route Line Overlay

Your full route drawn on the map with a glowing green line

Phone Notifications

Your texts, emails, whatever — they show up on your glasses

Voice Directions

TTS reads out your turns and routes audio to your glasses via Bluetooth A2DP

Tile Proxy

If glasses don't have Wi-Fi, phone fetches map tiles and sends them over Bluetooth

Save Places

Found a spot you like? Save it. Long-press to delete

Over-the-Air Updates

Push app updates to glasses over Bluetooth without cables or ADB

Three Layout Modes

Full, Corner, or Mini — tap to cycle through display layouts

Free & Open Source

OSRM

Routing engine with no API key required. Fast, accurate directions powered by open-source software.

Nominatim

Address search powered by OpenStreetMap. No API key, no rate limits, just free location search.

OpenStreetMap

Map data from the world's largest free and open geographic database (ODbL license).

CartoDB

Beautiful dark-themed map tiles (Dark Matter) with a green tint perfect for AR displays (CC BY-SA).

OSMDroid

Android map library that brings it all together with efficient tile management and rendering.

Open Source

Use it, modify it, build on it. The entire codebase is free and open for everyone.
